# Baseball Trivia

Play baseball trivia with friends! Features three game modes inspired by Jomboy Media.

## Game Modes

- **Trivia** - Multiple choice baseball questions (nicknames, stats, history)
- **Tic Tac Toe** - Answer questions to claim squares on a 3x3 grid
- **Pinpoint** - Name players, teams, and stats based on clues

## Deploy Your Own

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/doronschwartz/baseball-trivia)

## Run Locally

```bash
npm install
npm start
# Open http://localhost:3000
```

## How to Play

1. Pick a game mode
2. Enter your name
3. Share the 4-letter room code with friends
4. Host starts when everyone's in

## Tech Stack

- Node.js + Express
- Socket.io for real-time multiplayer
- MLB Stats API for live data
